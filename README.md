
# 📘 Diário de Bordo PWA

Aplicação Web Progressiva (PWA) desenvolvida com HTML, CSS e JavaScript puro para registrar atividades diárias. O projeto permite criar, visualizar e remover registros, funcionando também offline graças ao Service Worker.

---

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- Manifest.json
- Service Worker

---

## Funcionalidades

- Criar entradas com título, descrição e data.
- Listar todas as entradas cadastradas.
- Remover entradas.
- Armazenamento persistente utilizando LocalStorage.
- Funcionamento offline.
- Instalação como aplicativo (PWA).
- Interface responsiva para desktop e dispositivos móveis.

## Como executar

1. Abra o projeto no Visual Studio Code.
2. Instale a extensão Live Server.
3. Clique com o botão direito em `index.html`.
4. Selecione **Open with Live Server**.
5. A aplicação será aberta no navegador.
